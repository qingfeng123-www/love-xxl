## ❓ 常见问题

### 🔧 游戏使用问题

**Q: 游戏没有声音怎么办？**
A: 现代浏览器通常需要用户交互后才能播放音频。如果自动播放失败，请：
- 点击游戏界面任意位置激活音效
- 检查浏览器是否静音
- 确保音频文件完整存在

**Q: 可以自定义宝石图片吗？**
A: 当然可以！制作64×64像素或更大的PNG图片，按照gem0.png到gem8.png的命名放入images文件夹即可。

**Q: 如何重置游戏进度？**
A: 有两种方法：
- 清除浏览器的localStorage数据
- 按F12打开开发者工具，在控制台输入：`localStorage.clear()`

**Q: 游戏在手机上显示异常？**
A: 请尝试：
- 横屏游戏获得更好体验
- 使用最新版Chrome或Safari浏览器
- 确保网络连接稳定

**Q: 可以离线游戏吗？**
A: 完全可以！游戏是纯本地HTML文件，无需网络连接即可畅玩。

### 🎮 游戏策略问题

**Q: 如何获得更高分数？**
A: 
- 优先制造连锁消除
- 合理使用特殊宝石
- 观察全局，寻找最优路径
- 在自由模式中练习技巧

**Q: 冰块太难处理怎么办？**
A:

- 利用线消除宝石快速清理

## 🔧 技术说明

### 🌐 兼容性支持
- **浏览器支持**：Chrome 60+、Firefox 55+、Safari 12+、Edge 79+
- **操作系统**：Windows 7+、macOS 10.12+、iOS 12+、Android 5.0+

### ⚡ 性能优化
- **Canvas 2D渲染**：流畅的60FPS游戏体验
- **智能内存管理**：避免内存泄漏，长时间游戏无压力
- **资源预加载**：游戏启动时预加载所有资源

### 🔒 数据安全

- **隐私保护**：不收集任何用户信息

- **前端技术**：HTML5 + CSS3 + Vanilla JavaScript
- **图形渲染**：Canvas 2D API
